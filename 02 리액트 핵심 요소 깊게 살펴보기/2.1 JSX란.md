### Q1. 다음 코드를 개선해주세요. 그리고 그렇게 해야하는 이유를 설명해주세요. (경민)
```javascript
function TextOrHeading({
    isHeading,
    children,
}: PropsWithChildren<{ isHeading: boolean }>) {
    return isHeading ? (
        <h1 className="text">{children}</h1>
    ) : (
        <span className="text">{children}</span>
    )
}
```
